# SolixBar 0.4.0

## Deutsch

- Direkter, portabler SOLIX-Login ohne lokale Projektpfade oder manuelle Helper-Installation.
- Mail und Passwort werden ausschließlich in der macOS-Keychain gespeichert.
- Das Release enthält die benötigte Python-Laufzeit und SOLIX-Module.
- Kumulativer Gesamtertrag wird dauerhaft aus API-Wert beziehungsweise Messverlauf fortgeführt.
- Atomare, dateibasierte Verlaufs- und Ertragsspeicherung mit Migration vorhandener Daten.
- Weniger API-Aufrufe durch Tages-/Gesamtcache und automatisches Backoff bei Fehlern.
- Weniger unnötige UI-Arbeit für nicht sichtbare Zusatzfenster.
- Aktualisierte deutsch-/englischsprachige Homepage, Dokumentation und Screenshots.

## English

- Direct, portable SOLIX login without local project paths or manual helper installation.
- Email and password are stored exclusively in the macOS Keychain.
- The release includes the required Python runtime and SOLIX modules.
- Cumulative total yield is persisted from the API value or measurement history.
- Atomic file-based history and yield storage with migration of existing data.
- Fewer API requests through daily/total caching and automatic error backoff.
- Less unnecessary UI work for hidden auxiliary windows.
- Updated German/English homepage, documentation, and screenshots.

Download `SolixBar-0.4.0-macOS-arm64.zip`, unpack it, and move `SolixBar.app` to `Applications`.
